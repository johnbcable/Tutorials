Using @font-face with unicode-range

A subset font uses a different file for each unicode range. Unicode ranges can be a few specific characters, or entire scripts. In this case, we've split AbhayaLibre into two files: one containing a Latin subset of characters, and one containing a Sinhalese subset.

Related specification: CSS Fonts Module Level 4.

This sentence is an example of English-language text typeset using AbhayaLibre.

මමෙම වාක්යය අබායා ලිබ්ර් භාවිතා කරමින් සිංහල අකුරු වර්ගයේ උදාහරණයකි.
